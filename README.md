# mettasphere
Mettasphere is a 6R-method meditation tracker designed for "Tranquil Awareness" (MN 111). Unlike forced-focus apps, we prioritize the Relax and Re-smile steps to physically dissolve craving ($Taṇhā$). Features granular hindrance logs, "Secret Jhana" factor tracking, and Sati-sensitivity metrics.Smile, Relax, and Observe.
